### C2. Add canonical tags via layout metadata
**Impact:** Prevents duplicate content penalties  
**Effort:** 15 min  

In `app/layout.tsx`:
```ts
export const metadata: Metadata = {
  metadataBase: new URL('https://orderlay.com'), // replace with actual domain
  title: {
    default: 'Orderlay — Restaurant Management Software',
    template: '%s — Orderlay',
  },
  description: "...",
  alternates: {
    canonical: '/',
  },
};
```

Per-page canonical is auto-generated by Next.js when `metadataBase` is set.

      userAgent: '*',
      allow: '/',
      disallow: '/api/',
    },
    sitemap: 'https://orderlay.com/sitemap.xml',

}

### C5. Fix massive SVG files (PERFORMANCE — LCP killer)
**Impact:** LCP could be 30+ seconds on mobile  
**Effort:** 2–4 hours  

Files to fix (all in `/public/asset/`):
- `owner.svg` (31 MB) → optimize with SVGO or replace with WebP
- `manager.svg` (24 MB) → same
- `kitchen.svg` (21 MB) → same
- `member.svg` (18 MB) → same

```bash
# Install SVGO
npm install -g svgo

# Optimize each SVG
svgo owner.svg -o owner.min.svg
```

If the SVGs contain complex illustrations that can't be simplified, convert to WebP:
```bash
# Using Inkscape or similar
inkscape owner.svg --export-type=png --export-filename=owner.png -w 1200
cwebp -q 80 owner.png -o owner.webp
```

Then serve WebP with PNG fallback via Next.js `<Image>`:
```tsx
<Image src="/asset/owner.webp" alt="Restaurant owner dashboard" width={600} height={400} />
```

### H3. Add FAQ schema to homepage and /restaurant-goer
**Impact:** FAQ rich results in SERP (accordion expansion)  
**Effort:** 45 min  

```tsx
// In app/page.tsx — add below the component
export function FAQSchema({ faqs }: { faqs: {question: string, answer: string}[] }) {
  return (
    <script
      type="application/ld+json"
      dangerouslySetInnerHTML={{
        __html: JSON.stringify({
          "@context": "https://schema.org",
          "@type": "FAQPage",
          "mainEntity": faqs.map(f => ({
            "@type": "Question",
            "name": f.question,
            "acceptedAnswer": { "@type": "Answer", "text": f.answer }
          }))
        })
      }}
    />
  )
}
```
